# egov-social-insurance-zip-organizer

e-Govの社会保険関連ZIPを、種類と年月で整理して展開するPowerShellスクリプトです。

## 概要

e-Govで電子送達される社会保険関連のお知らせZIPを、種類ごとに判定し、分かりやすい名前のフォルダに展開します。

このスクリプトは、作者が実際に利用している条件・環境向けに作成したツールを公開したものです。その他の条件・環境での動作は確認していません。

## 背景

社会保険関連のお知らせを郵便から電子送達に切り替えると、紙の郵便物を待たずに済む一方で、e-Govからダウンロードされるファイルは機械的な名前のZIPファイルになっています。

そのまま保存すると、あとから見返したときに、

* 何の通知なのか分かりにくい
* 何年何月分なのか分かりにくい
* ZIPを開いて中身を確認しないと分類できない
* ファイル名が長く、整理しづらい

という問題があります。

このスクリプトは、ダウンロードしたZIPファイルをドラッグ＆ドロップするだけで、ZIP内のXMLファイルを読み取り、通知の種類と年月を判定して、分かりやすい名前のフォルダに展開します。

## できること

ZIPファイル内を検索し、以下のような社会保険関連ファイルを判定します。

* 被保険者データ
* 保険料納入告知額・領収済額通知
* 社会保険料額情報

判定後、例えば次のようなフォルダ名で展開します。

```text
令和07年10月分被保険者データ
令和7年12月送付分納入告知額・領収済額通知
令和8年3月分社会保険料額情報
```

## セットアップ
`Extract-JPSZipFolders.bat` と `Extract-JPSZipFolders.ps1` を、Windows上の同じフォルダに置きます。

`Extract-JPSZipFolders.ps1` の冒頭にある展開先のフォルダのパスの定義を自分の環境に合わせて書き換えてください。初期では以下のようになっています。
```
$OutputPathPattern1 = "\\server\share\被保険者データ"
$OutputPathPattern2 = "\\server\share\納入告知額・領収済額通知"
$OutputPathPattern3 = "\\server\share\年金\社会保険料額情報"
```

必要に応じて、`Extract-JPSZipFolders.bat`のショートカットをドラッグ＆ドロップ操作をする場所に作成してください。

## 使い方

e-GovからダウンロードしたZIPファイルを `Extract-JPSZipFolders.bat` にドラッグ＆ドロップすると処理を開始します。

（`bat` ファイルのショートカットを作成しておけば、スクリプト本体を置いたフォルダ以外でもドラッグ＆ドロップで実行できます）

スクリプトはZIP内の階層を調べ、該当するXMLファイルが見つかった場合、展開先とフォルダ名を表示します。

展開してよいか確認してから実行するため、意図しない展開を避けられます。

## ZIPの展開先

通知の種類ごとに、あらかじめ指定したフォルダへ展開できます。

指定したフォルダが存在しない場合や、ネットワークドライブの切断などで書き込みできない場合は、Desktopへ展開します。

展開に成功した場合は、最後に展開先の親フォルダをExplorerで開きます。

## 付属ツール

横長で読みにくいCSVファイルを確認するための補助ツールとして、`csv_vertical_ascii.bat` を同梱しています。

`csv_vertical_ascii.bat` と `csv_vertical_ascii.ps1` を、Windows上の同じフォルダに置いておき、CSVファイルを `csv_vertical_ascii.bat` にドラッグ＆ドロップすると、その内容をテキストで縦方向に表示します。

## 注意事項

このスクリプトは、e-Govで電子送達される社会保険関連ZIPのうち、作者が確認した形式や条件をもとに仕様を推測して作成したものです。

確認範囲は、自社1社宛てに届いたおよそ半年分のファイルです。

企業や事業所の条件が異なる場合、またはe-Gov側のファイル形式やXML構造が変更された場合、正しく判定できない、または動作しない可能性があります。

社会保険関連の通知には、個人情報や事業所情報が含まれる場合があります。展開先フォルダのアクセス権限、バックアップ、共有設定、取り扱いには十分注意してください。

## 動作環境

* Windows
* PowerShell
* .NET標準のZIP処理機能
